# BeyondBoundary

T20 World Cup Fantasy Cricket Prototype.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure

- `app/`: Next.js App Router source.
- `app/(public)`: Public pages (Landing).
- `app/(auth)`: Auth pages (Login, Signup).
- `app/(dashboard)`: Authenticated pages (Dashboard, Profile, etc.).
- `stitch-prototype/`: Original Stitch source files.
